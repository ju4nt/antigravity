# 🚀 Landing Page B2B — Colconexus Datacenter SAS

Página web corporativa de alta conversión para Colconexus Datacenter SAS, una agencia especializada en Inteligencia Artificial, Telefonía VoIP y Desarrollo de Software a medida.

## ✨ Características

- **Fondo 3D interactivo**: Partículas con profundidad real (eje Z) que reaccionan al movimiento del mouse y se conectan entre sí formando redes de fibra óptica.
- **Diseño Glassmorphism**: Tarjetas semitransparentes con blur, borde sutil y gradientes neón azul/púrpura.
- **Animaciones Framer Motion**: Elementos que aparecen con suavidad al hacer scroll.
- **Integración WhatsApp**: Todos los botones de conversión abren WhatsApp con mensajes pre-redactados según el servicio de interés del cliente.
- **SEO completo**: Metadatos, OpenGraph, lang=es, html semántico.
- **Responsivo**: Diseño adaptado para Móvil, Tablet y Desktop.

## 🛠️ Stack Tecnológico

| Tecnología | Uso |
|---|---|
| **Next.js 16** | Framework React con SSR/SSG |
| **TailwindCSS 4** | Estilos utilitarios modernos |
| **Framer Motion** | Animaciones de entrada y hover |
| **Lucide React** | Iconos minimalistas |
| **Canvas API pura** | Fondo de partículas 3D interactivas |

## 📦 Instalación y Desarrollo

```bash
# Clonar el repositorio
git clone https://github.com/ju4nt/antigravity.git

# Navegar a la carpeta
cd 01_paginas/landing_page_b2b

# Instalar dependencias
npm install

# Levantar servidor de desarrollo local
npm run dev
```

La página estará disponible en **http://localhost:3000**

## 🚀 Despliegue Gratuito en Vercel

1. Ve a **[vercel.com](https://vercel.com)** e inicia sesión con tu cuenta de GitHub.
2. Haz clic en **"Add New Project"**.
3. Selecciona tu repositorio `antigravity`.
4. En **Root Directory**, escribe: `01_paginas/landing_page_b2b`
5. Haz clic en **Deploy** ✅

Vercel detectará automáticamente que es un proyecto Next.js y lo publicará de manera **100% gratuita** en una URL tipo `https://tu-proyecto.vercel.app`.
