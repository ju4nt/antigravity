# 🚀 Ecosistema de Riqueza Digital con OpenClaw IA

Este repositorio contiene la arquitectura, configuraciones y perfiles de agentes de IA autónomos diseñados para automatizar múltiples modelos de negocio digitales y flujos de trabajo corporativos para **Colconexus**.

## 🧠 Estructura del Ecosistema (Fase 1)

El sistema utiliza **OpenClaw** como middleware de orquestación y **Groq (Llama 3.3 70B)** como motor de inferencia gratuito de alta velocidad.

### 👥 El Equipo de Agentes (7 Bots)

*   **🎬 Director_YT**: Orquestador de canales de YouTube automáticos.
*   **🔍 Alpha_Researcher**: Cazador de tendencias de alto RPM en Reddit y Google.
*   **🎯 Affiliate_Master**: Especialista en cierre de ventas y copys para Hotmart.
*   **🛠️ Utility_Builder**: Arquitecto de micro-aplicaciones web generadoras de ingresos.
*   **🏗️ Dev_QA**: Seguridad y calidad de código para el ERP y bots.
*   **🎨 Creative_Studio**: Generador de anuncios, reels, clips y visuales.
*   **📊 CFO_Monitor**: Vigilante del ROI y rentabilidad financiera.

## 📂 Contenido del Repositorio

*   **`sistema_agentes/`**: Contiene los archivos `.md` con los perfiles, de identidad y prompts profundos de cada agente.
*   **`DEPLOYMENT_DEBUG.md`**: Historial detallado de errores y soluciones técnicas durante el despliegue.
*   **`python_code/`**: Scripts de soporte para integración y automatización.

## 🛠️ Instalación y Configuración

1.  Clonar el repositorio: `git clone [url-del-repo]`
2.  Tener instalado Node.js y OpenClaw (`npm install -g openclaw`).
3.  Configurar las API Keys de Groq y Telegram en el archivo `.openclaw/openclaw.json` (usar `openclaw.json.example` como guía).
4.  Lanzar el motor: `npx openclaw gateway`.
